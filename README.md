# AI Website Usability Analyzer

A full-stack AI web application that evaluates website usability using heuristic analysis and automated scoring.

## 🔗 Live Demo
[View Live App](https://ai-website-usability-analyzer.onrender.com/)

## 📌 About
This tool analyzes any website's usability by examining its structure, navigation, and responsiveness. It generates an automated usability score along with actionable improvement suggestions to help developers build better user experiences.

## ✨ Features
- Heuristic-based usability analysis
- Automated scoring across multiple usability parameters
- Analyzes website structure, navigation, and responsiveness
- Generates actionable improvement suggestions
- Clean interactive interface for real-time analysis

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python
- **AI/ML:** Heuristic analysis engine
- **Deployment:** Render

## 🚀 How to Run Locally
```bash
git clone https://github.com/poojitha1603/ai-website-usability-analyzer.git
cd ai-website-usability-analyzer
pip install -r requirements.txt
python app.py
```
